# 🌌 React Jobs — Ultra Premium UI/UX Edition

An Awwwards-quality, cinematic job platform for React developers and tech teams. This project elevates the default developer job board template into an immersive, premium workspace experience with rich motion design, futuristic LED matrix typography, and glassmorphic layouts, while keeping **100% of the core functionality, routing, state, and validation exactly the same**.

---

## 🌟 Key Enhancements

### 1. 🎬 Cinematic Hero Section
*   **Slow-Motion Cinematic Workspace Video:** Replaced the hero background with a slow-playback futuristic background.
*   **Holographic Right Panel:** Interactive card playing a high-quality video showing a workspace with floating React and holographic components.
*   **Layered Composites:** Custom overlay gradients, glass blur, soft vignetting, and ambient glow orbs to maintain supreme text readability.
*   **Floating Particles:** Interactive, floating particle system reflecting color tokens (`#D4FF00` and `#A5CCFF`).

### 2. 🔤 Futuristic Developer Typography
*   **Google Fonts Integration:** Linked **Doto** (LED Matrix display font), **Space Grotesk** (tech headings), and **Inter** (high readability body text).
*   **Typographical System:**
    *   *Hero Heading & Large Stat Numbers:* Doto (LED pixel style with custom neon glow shadows).
    *   *Section Titles:* Space Grotesk Bold.
    *   *Body & Labels:* Inter.
    *   *Buttons & Navigation:* Inter SemiBold.

### 3. 🃏 Interactive Glassmorphic Job Cards
*   **3D Mouse-Move Tilt:** Smooth magnetic cards that tilt and follow the user's cursor.
*   **Interactive Glow:** Dynamically changes bottom glow from secondary soft blue to neon lime-yellow on hover.
*   **Company Avatar Generator:** Replaced generic briefcase icons with elegant, custom company initial avatars matching the theme.

### 4. 🧭 Scroll-Aware Glass Navbar
*   **Linear/Vercel Style Hide/Show:** Dynamically hides when scrolling down fast to save screen space, and slides back into view when scrolling up.
*   **Glass Morphism:** Blur level and background opacity saturate dynamically as you scroll.
*   **Animated Active Link Indicators:** Smooth Framer Motion layout pill that transitions between navigation options.

### 5. 📋 Optimized 2-Column Compact Forms (Add/Edit Job)
*   **No Excessive Scroll:** Respaced and rearranged layout elements into a responsive 2-column grid.
*   **Compact Areas:** Textareas resized to keep the entire form viewable on a single desktop viewport.
*   **Glass Inputs:** Inputs glow and focus with neon lime rings and custom shadows.

## ⚙️ Running Locally

### Prerequisites
*   Node.js (v18+ recommended)
*   npm

### Installation

1. Clone this repository (if not already local):
   ```bash
   git clone https://github.com/aryansingh7877/ReactDev.git
   cd ReactDev/ReactDev
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
